    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1><a id="user-content-c-for-image-processing" class="anchor" href="#c-for-image-processing" aria-hidden="true"><span class="octicon octicon-link"></span></a>C++ for Image Processing</h1>

<p>This code is part of the coursework for C++ for Image Processing course. It involves writing an image processing program in c++.</p>

<h3><a id="user-content-main-functionalities" class="anchor" href="#main-functionalities" aria-hidden="true"><span class="octicon octicon-link"></span></a>Main functionalities:</h3>

<h5><a id="user-content--loading-image-and-basic-filtering" class="anchor" href="#-loading-image-and-basic-filtering" aria-hidden="true"><span class="octicon octicon-link"></span></a>* Loading image and basic filtering:</h5>

<p>-loads and displays an image, pixel value rescaling and shifting; </p>

<p>-performs a convolution on the image forsmoothing the image; </p>

<p>-edge detection in the image; </p>

<p>-performs order statistic filtering; </p>

<p>-performs point processing;  </p>

<h5><a id="user-content--improvingaltering-the-image-contrast" class="anchor" href="#-improvingaltering-the-image-contrast" aria-hidden="true"><span class="octicon octicon-link"></span></a>* Improving/altering the image contrast:</h5>

<p>-thresholding; </p>

<p>-histogram equalisation;  </p>

<h5><a id="user-content--selecting-a-region-of-interest-roi-performing-the-image-processing-function-on-the-roi" class="anchor" href="#-selecting-a-region-of-interest-roi-performing-the-image-processing-function-on-the-roi" aria-hidden="true"><span class="octicon octicon-link"></span></a>* Selecting a Region of Interest (ROI). Performing the image processing function on the ROI.</h5>

<h5><a id="user-content--menu-system-ability-to-perform-a-combination-of-image-processing-functions-on-the-roi-including-an-undo-function" class="anchor" href="#-menu-system-ability-to-perform-a-combination-of-image-processing-functions-on-the-roi-including-an-undo-function" aria-hidden="true"><span class="octicon octicon-link"></span></a>* Menu system. Ability to perform a combination of image processing functions on the ROI, including an "undo" function.</h5>

<h4><a id="user-content-how-to-run-the-program" class="anchor" href="#how-to-run-the-program" aria-hidden="true"><span class="octicon octicon-link"></span></a>How to run the program:</h4>

<p>If you want to explore the project from your own machine, then g++ and wxwindows are easy to install under linux. To run the program, simply type ./window in your command line once you are in the correct directory. If you must use windows, cygwin is one option - it comes with gcc/g++ and wxwindows does work as well.Alternatively, it is very easy installing a dual boot linux machine at home. If you want to alter the code, use the Make file to compile the source into a runnable object and run it to start the program.</p>
